# java-racingcar
자동차 경주 게임 미션 저장소

## 👑 구현할 기능 목록
- **경주에 참여할 n대의 자동차 이름을 입력받는 기능**
    - 자동차 이름은 쉼표로 구분한다.
    - 자동차 이름은 5자 이하다.
    - [예외] 자동차 이름이 5자보다 길 경우 
  
- **몇 번의 이동을 할 것인지 입력받는 기능**
    - [예외] 입력받은 이동 횟수가 0이하일 경우

- **주어진 횟수만큼 자동차들을 이동시키는 기능**
    - 0~9 사이 랜덤 숫자를 선정한다. 4이상이면 전진 3이하면 멈춘다.
    - 매 이동 횟수 마다 자동차 이름과 거리를 출력한다.

- **게임 완료 후 우승자 출력하기**
    - 우승자는 한 명 이상일 수 있음
